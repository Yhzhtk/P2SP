Util4J
======

Java 一些基础工具类的封装。方便开发效率
-----

### 包括以下内容
* 算法，如base64，UrlConvert等
* IO，读写文本文件，读写xls等
* 邮件发送接收
* 图片压缩处理
* ZIP文件压缩解压
* JSON数据处理
* HttpClient封装
* 正则基础类
* 数据库连接池
* 其他
